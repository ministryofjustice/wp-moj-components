Please note; there are 3 levels of monitor:
  1. Out of Date Software
  2. Security Vulnerabilities
  3. Abandoned Software

 

Out of Date Software

Our monitor may have found that WordPress or an installed plugin is not the latest version. When this happens is it recommended to bring the software installation up to the latest version to prevent any potential security holes.

 

Security Vulnerabilities

This monitor discovers active issues with our installed assets and produces various bits of data:
  • The type of vulnerability found
  • Whether or not the vulnerability is fixed, and if so
  • The version number of the fixed software
It is recommended to investigate the vulnerability ASAP.
  1. Can we upgrade to a fixed version? Or,
  2. Should we consider a replacement?

 

Abandoned Software

From time-to-time some third-party software can become so aged that inherently they become insecure. For instance; a plugin that is more than 12 months old is likely to have missed 3-4 WordPress releases including a major release.  The recommendation for abandoned software is to find a replacement ASAP.
